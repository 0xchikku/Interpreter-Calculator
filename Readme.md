# Interpreter Calculator
---

This Calculator bot will help you to do calculations as in **Discord Server** channels.

### Command : `$ic`

## How it Works
Just use the command `$ic` followed by `math expression`

Eg:
>`$ic 2+2` 

> `Result is 4`
